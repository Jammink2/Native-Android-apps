A run-once (Android 3.1 or greater) binary for finding the battery level, voltage, temperature, technology, status, health, and plugged state of an Android-based phone.
  
Create a new Eclipse Android Project, main package named "com.hammink.BatteryLevel"
create "/res/drawable-hdpi/my_border.xml" and paste the contents of this my_border.xml.  Do the same for "res/layout/main.xml"
  
Working binaries of this app can be found at:
http://people.mozilla.com/~jhammink/android_native_battery_app/bin/

BatteryLevel.apk - for tablets (Android 3.1 or better)

PhoneBatteryLevel.apk - for phones (Android 2.1 or better)
